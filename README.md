# Программа для хранения записей о специальности университета

1. Программа поддерживает до 100 записей (можно регулировать в исходном коде)
2. Есть возможность сохранять запись в файл, который будет закодирован
3. Есть возможность удалять/редактировать записи
4. Каждая запись имеет уникальный ID
5. Записи сортируются по названию специальности, затем по ВУЗу, затем по конкурсу (т.е. если название специальности и ВУЗ одинаковы, сортировка идет по конкурсу)
6. Присутсвтует валидатор в некоторых строках

Для запуска нужен Qt Creator 4.11.1 (Community). Открываем .pro файл и компилируем.
Для хранения записей используется QHash с типом итератора Java.
